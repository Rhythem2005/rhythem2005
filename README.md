### [🧠 Nova SLM](https://github.com/Rhythem2005/NOVA-SLM)

**30M parameter GPT-style language model, trained from scratch.**

Decoder-only transformer with 6 layers, 6 attention heads, 384-dim embeddings, flash attention, pre-LN normalization, weight tying, and cosine LR decay with linear warmup. Trained on 2.1M TinyStories using PyTorch on Apple Silicon with bfloat16 mixed precision. No pretrained weights — every parameter learned from random init.

`PyTorch` `tiktoken` `bfloat16` `Transformers` `From Scratch`

### 🍕 FlavorFind

**AI-powered food discovery engine.**

Natural language food search that ranks results across delivery platforms. Gemini handles query understanding, Playwright scrapes real-time listings, and FastAPI serves the backend. Built to solve the "what should I eat" problem with actual intelligence behind the search.

`React` `FastAPI` `Gemini API` `Playwright`
